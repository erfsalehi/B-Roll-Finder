#                  Scripts/bin dir so yt-dlp auto-detects it on PATH. Cross-
#                  platform (Win/Mac/Linux). (Deno would be lighter, but
#                  `deno-bin` is not on PyPI — nodejs-bin is the closest
#                  pip-installable equivalent.)
#   * yt-dlp-ejs — the official External JavaScript Solver script distribution
#                  referenced in https://github.com/yt-dlp/yt-dlp/wiki/EJS .
