Analyze the following video script and timestamps. Identify key moments worth highlighting on screen to keep the audience engaged. 

Categories to look for:
- HEADINGS/TITLES: Look for major topic shifts, chapter markers, or enumerated points (e.g., "1. Introduction", "2- Strategy"). If the script uses numbers before titles, PRESERVE THEM exactly as written.
- MONEY/PRICING: Mention of specific dollar amounts or financial gains/losses.
- STATISTICS: Important numbers, percentages, or data points.
- CORE CONCEPTS: Powerful phrases that summarize a key takeaway.

Constraint:
- highlight_text MUST be short (under 5 words).
- For HEADINGS, use the exact text from the script if it marks a new section, including any numbers (e.g. "1. OVERVIEW").
- start_time and end_time MUST be in HH:MM:SS,mmm format if provided in the script, or seconds.
- IMPORTANT: if the input is a TIMESTAMPED TRANSCRIPT (lines like "[12.345 - 14.890]: text"), the timestamps are the real audio timeline including silences. Use them VERBATIM — pick start_time from the first segment that contains the highlighted phrase and end_time from the last segment that contains it. Do NOT estimate from word count; the natural pauses between segments are real silences in the voiceover and must be respected.
- original_caption MUST be the relevant snippet from the script.
- suggested_sfx: Suggest a 1-2 word sound effect that fits the context (e.g., "whoosh", "ding", "cash register", "pop", "camera shutter").

Return a JSON array of objects under a top-level "highlights" key.
Example:
{
  "highlights": [
    {
      "start_time": "00:00:15,000",
      "end_time": "00:00:18,500",
      "original_caption": "1. We managed to save over ten thousand dollars on this engine rebuild.",
      "highlight_text": "1. $10,000 SAVED",
      "category": "money",
      "suggested_sfx": "cash register"
    }
  ]
}
